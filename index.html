HTML

  • The head element contains information about the webpage
  • The body element represents the visible content shown to the user
  • ul means unordered lists
    • ol means ordered lists il means listed items

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • * is used to apply to all elements
  • Selector defines the element that the rules apply to
  • CSS box model

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • git checkout main: switches to branch "main"
  • git add -A: add all the changes in the current working branch
  • git commit -m "commit message": describes what the commit contains
  • git pull origin main: receives modifications into the local environment
  • git push origin: pushes changes locally to the remote GitHub branch

JavaScript

  • A variable is a named container that allows us to store data in our code
  • ; at the end of a line indicates where the statement ends
  • operator - math symbol that produces results based on two values
  • var is used to declare variables
  • array means a single variable thats used to hold a group of date
  • null is a special value that refers to the absence of a value
